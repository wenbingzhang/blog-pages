## 创建标签
```bash
git tag <tagname>
```

## 推送标签到远程仓库
```bash
git push origin <tagname>
```

## 推送所有标签到远程仓库
```bash
git push origin --tags
```

## 查看本地所有标签
```bash
git tag
```

## 查看远程所有标签
```bash
git ls-remote --tags origin
```

## 删除本地或远程标签
```bash
git tag -d <tagname>

# 删除远程标签
git push origin :refs/tags/<tagname>
```
